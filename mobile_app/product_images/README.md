# Ảnh sản phẩm (Product images)

Thư mục này dùng cho ảnh sản phẩm để tránh lỗi 404 path trùng (assets/assets/) trên Flutter Web.

**Bắt buộc:** `default.jpg` — ảnh mặc định khi không có ảnh theo tên.

**Thêm ảnh theo tên:** Đặt file jpg/webp tại đây (max ~600px, <400KB). Ví dụ: tao_my.jpg, chuoi.jpg, cam.jpg, xoai.jpg, dua_hau.jpg, ca_chua.jpg, …
